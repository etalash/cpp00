# CPP00 – First C++ Project at 42 School

This repository contains my **CPP00** project, which was my very **first C++ project** at 42 School. The project focuses on introducing basic C++ syntax, understanding program structure, and writing simple programs.

---

## Overview

`CPP00` is designed to teach the fundamentals of C++ programming, including variables, functions, basic input/output, and simple data manipulation.

### Features

* Writing simple C++ programs
* Understanding **basic syntax** and program structure
* Introduction to functions and variables
* Compilation and execution of C++ code

### Installation & Build

```bash
git clone <repo-url>
cd cpp00
make
./<executable>
```

### Notes

* This was my **first official C++ project**, marking the start of my C++ learning journey.
* Focused on mastering the basics before moving to object-oriented programming in later modules.
